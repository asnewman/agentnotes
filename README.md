# AgentNotes

A local-first knowledge base with CLI interface. All notes stored as markdown files with YAML frontmatter for metadata. Simple, portable, human and AI-agent readable.

## Commands

```
agentnotes add <title>       Create a new note
agentnotes list              List notes with filters
agentnotes show <id>         Display a note
agentnotes search <query>    Full-text search
agentnotes edit <id>         Edit in $EDITOR
agentnotes delete <id>       Delete a note
agentnotes tags              List all tags
agentnotes cat <id>          Raw markdown output
```
