Custom Text Editor

A vanilla JS text editor with externally-managed state

How to use

  • Click in the editor to position your cursor
  • Type to insert text
  • Click and drag to select text
  • Double-click to select a word
  • Use the toolbar buttons to apply decorations to selected text
  • Use arrow keys to navigate, Shift+arrows to extend selection
  • Cmd/Ctrl+A to select all
